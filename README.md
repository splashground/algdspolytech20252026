Лабораторная работа  
Вариант 25

Задача упаковки в рюкзак

Условие задачи

Задано конечное множество предметов U.  
Для каждого предмета u ∈ U известны:
- s(u) — размер (вес) предмета
- v(u) — стоимость предмета

Также заданы:
- B — вместимость рюкзака
- K — минимально допустимая суммарная стоимость

Необходимо определить, существует ли такое подмножество U' ⊆ U, что:

∑ s(u) ≤ B  
∑ v(u) ≥ K  

Если такое множество существует, требуется вывести номера предметов,
входящих в U'. Нумерация предметов начинается с 1.  
Если решения не существует — вывести 0.

Формат входных данных

В текстовом файле input.txt хранится:

- В первой строке три целых положительных числа: |U|, B, K
- Во второй строке |U| целых положительных чисел — размеры предметов
- В третьей строке |U| целых положительных чисел — стоимости предметов

Ограничения:
- |U| < 1000
- B < 2³²
- K < 2³²

Формат выходных данных

В текстовый файл output.txt выводится:

- номера предметов, входящих в подмножество U'
- или число 0, если решения не существует

Алгоритм

Задача решается с помощью:

- Динамического программирования
- Решения задачи 0/1-рюкзака
- Поиска допустимого множества с суммарной стоимостью не менее K

Используется массив dp[w], где dp[w] — максимальная стоимость,
достижимая при суммарном весе w.

Сложность алгоритма

- Временная сложность: O(|U| · B)
- Пространственная сложность: O(B)

Особенности реализации

- Реализация выполнена на языке C
- Используется одномерное динамическое программирование
- Хранится информация для восстановления выбранных предметов
- Корректность проверена с помощью модульных тестов Google Test
- Код отформатирован в стиле Google
